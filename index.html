EAN

EAN

Det här är ett program som jag utvecklat för att förenkla hanteringen av Streckkoder på Ica Kvantum i Piteå.

Bakgrund

I och med självscanningssystemen som är vanliga idag är man tvungen att ha streckkoder på alla varor så att kunderna kan registrera dessa själva. I regel sätter man upp streckkoder vid varorna om det inte är förpackade varor som har streckkoder på förpackningen. På grund av begränsningar i kassasystemen så innebär det att samma streckkoder även måste finnas i kassorna för att man på korrekt sätt ska kunna stämma av kundernas köprundor. För att förenkla för kassörerna, som ofta kan vara relativt nya medarbetare, bör dessa listor vara lättöverskådliga och gärna med bilder. Tidigare hantering av detta har skett i word-dokument och hanterats manuellt vilket bland annat är:

  • Tidsödande
  • Lätt att bli fel

På grund av sortimentsförändringar över tid krävs det även att dessa listor underhålls kontinuerligt där man tar bort gamla streckkoder och lägger till nya.

Mitt program går ut på att man kan ladda en csv-fil som innehåller data kring ett sortiment av varor, inkluderat varor som inte för närvarande finns i butiken. Man kan sedan enkelt välja vilka som ska finnas med på listan och sedan skriva ut en lista. Man kan även spara ändringarna så att man i framtiden enkelt kan justera listan. På grund av att IT-systemen är låsta och endast program som är godkända centralt i regel kan installeras på datorerna valde jag att bygga programmet som en ren front-end webb-applikation. Det betyder att filhantering sker med hjälp av HTML5s filsystem och on-the-fly generering av filer i webbläsaren som sparas genom nerladdning.

Besök gärna projektets github repo för att se hur jag löst problemen!
